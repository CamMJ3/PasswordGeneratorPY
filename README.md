# Password Generator Script in Python

Secure password generator using Python's secrets module, featuring a customizable length and a user-friendly menu interface.

## Characteristics

- **High Security**: Leverages the `secrets` module for cryptographically strong random numbers.
- **Full Customization**: User-defined password length.
- **Universal Symbols**: Combines ASCII letters, digits, and punctuation.

## Project's structure
- main.py
- passwords.py

## Execution
Open your terminal, and run...
"**main.py**" with the version of python installed in your device.
